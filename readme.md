# Veritas — Data Research & Dashboard (Submission)

Project submission by Assaf Josef Berman for the Veritas team at Δ.

This repository contains a short data-research exploration and a visual dashboard created as part of the assessment. The work focuses on network traffic analysis, exploratory data analysis, basic modelling, and interpretability visuals.

## What's included
- [`data_research.ipynb`](data_research.ipynb) — Primary analysis notebook. Loads the dataset, runs SQL-style queries (via SQLite), performs EDA, protocol and attack analyses, model training, and interpretability plots.
- [`Veritas-Data-Set.csv`](Veritas-Data-Set.csv) — The dataset used for the analysis.
- [`dashboard.jpg`](dashboard.jpg) — Static snapshot of the dashboard produced for the assignment using Tableau.
